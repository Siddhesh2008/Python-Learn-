# Python Practice Repository 🐍

Welcome to my Python Practice Repository! This repository contains organized Python programs, exercises, and projects created while learning and mastering Python programming.

## 📌 Repository Structure

The codebase is structured into **9 organized modules**, progressing from basic syntax to advanced topics, GUI development, and projects:

### 📁 [01_basics](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/01_basics)
Fundamentals of Python programming.
- **Variables & Datatypes**: Printing, basic variable assignments, casting, and data type checks.
- **Operators**: Assignment operators, logical operators, and membership operators.
- **Basic Math**: Arithmetic operations, math functions, and generating random numbers.
- **Strings**: Essential string methods and formatting.

### 📁 [02_control_flow](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/02_control_flow)
Decision-making and loops.
- **Conditionals**: Basic `if-else` statements, conditional expressions, match-case structure, and `name_main` block.
- **Loops**: For loops, while loops, and nested loops.

### 📁 [03_collections](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/03_collections)
Working with collections and iterables.
- List, Tuple, Set, and Dictionary implementations.
- List/Set/Dictionary comprehensions, 2D lists, list indexing, and iterables.

### 📁 [04_functions](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/04_functions)
Modular code structures.
- Function definitions and variable scopes.
- **Arguments**: Keyword arguments, default arguments, and arbitrary arguments (`*args`, `**kwargs`).

### 📁 [05_oop](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/05_oop)
Object-Oriented Programming principles.
- Classes, objects, constructor initialization, and class/instance variables.
- Inheritance (single, multiple, multi-level), `super()`, polymorphism, duck typing, and decorators (including `@property` and `@classmethod`).

### 📁 [06_file_handling](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/06_file_handling)
File operations.
- File existence detection, reading text files, and writing data to files (Text, JSON, and CSV format).

### 📁 [07_advanced](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/07_advanced)
Advanced concepts and libraries.
- Exception handling, multithreading, date & time formatting, custom modules, and REST API consumption.

### 📁 [08_gui_pyqt5](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/08_gui_pyqt5)
Desktop Graphical User Interfaces.
- PyQt5 widgets, buttons, checkboxes, text fields (LineEdit), radio buttons, styling with stylesheets, layouts, and image displays.

### 📁 [09_assignments](file:///C:/Users/Siddhesh/Desktop/coding/Python-Learn-/09_assignments)
Mini-projects, games, and practical exercises.
- Classic games: Hangman, Rock-Paper-Scissors, Quiz Game, Dice Roller, Slot Machine.
- Utilities: Alarm Clock, Digital Clock, Stopwatch, Calculator, Temp Converter, Encrypter/Decrypter, user input validator, and weight converter.
